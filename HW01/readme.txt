Сборки программы: gcc main.c -Wall -Wextra -Wpedantic -std=c11
Программа main.c проверяет файл в формате JPEG на наличие скрытого архива в формате ZIP без пароля с файлами в корневой директории.
Программа main.c обрабатывает файл test.jpg, содержащий одно изображение в формате JPEG, и выводит результат проверки.
Программа main.c обрабатывает файл zipped.jpg, содержащий одно изображение в формате JPEG и архив в формате ZIP, и выводит результат проверки и имена файлов архива.
Программа проверена в системе Ubuntu x64.
